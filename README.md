# React + TypeScript 프로젝트

이 프로젝트는 [Create React App](https://github.com/facebook/create-react-app)을 기반으로 만들어졌으며, `React 19`, `TypeScript 5.8.3`, `Yarn`을 사용합니다.

## 🛠 사용 기술

- React 19.1.0
- TypeScript 5.8.3
- Styled-Components
- React Router
- Yarn 1.x

## 📦 사용 가능한 명령어

### `yarn start`

개발 서버 실행 (http://localhost:3000)

### `yarn build`

프로덕션용 빌드 생성
